mini project by daniel
tag name description
<form> Form used to collect and send information to a source
<input> Input used to render input fields on a document such as buttons, checkboxes and text fields
<label> Label used to render labels next to inputs such as to provide information
<select> Select used to create a dropdown list (a kind of input)
<option> Option used to make options for dropdown fields
<datalist> Datalist paired together with an input for basic search/autocomplete. used in combination with <option>

displaying html tags as text

When displaying tags as text, you cannot use the actual tag symbols. you need to use < and > to render the start and end tags respectively.
