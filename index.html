ibada.me

This is my portfolio website (what you're on right now!).

Originally I tried making it with a css library (w3.css) but it ended up keeping me from understanding css layouts with flexbox properly, so I ended up scrapping it.

My second iteration of this website was a site where I tracked my favourite artists' latest singles and albums and, although it didn't look pretty, I was proud to have made it.

Right now, this website is a hub for my projects and an exercise on proper CSS.

credit card checker

This was my first project with an actual algorithm (the Luhn algorithm) and was a lot of fun because it's a real-life application of the algorithm!

I suggest checking it out on my github since you can see a few of the credit card numbers I split into arrays and how the algorithm is implement using a for loop and if/else statements.

robotics for all

I've worked a little bit on the Robotics for All website redesign that was done in React (preact and bootstrap with vite) and while it won't be done anytime soon, it's was a good learning experience with how to make an already-working website properly responsive.

I've also worked on small javascript applications that transmit data to and from slack/google sheets! One of these displayed open classes and students while the other was a way to find substitute teachers for classes that were missing a teacher.
