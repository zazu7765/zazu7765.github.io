credit card checker

This was my first project with an actual algorithm (the Luhn algorithm) and was a lot of fun because it's a real-life application of the algorithm!

I suggest checking it out on my github since you can see a few of the credit card numbers I split into arrays and how the algorithm is implement using a for loop and if/else statements.
